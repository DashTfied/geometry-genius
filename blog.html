What is DOM and what is the purpose of DOM?

Answer: DOM, Document Object Model is interface for website which is used to update, manipulate the site. We can make changes on website using DOM.

How will you select HTML elements using DOM? Name the DOM methods.

Answer: Elements can be selected by using following DOM methods, getElementByID, getElementsByClassName, getElementsByTagName, querySelector, querySelectorAll.

What is event bubble?

Answer: When an element receives an event and keeps transmitting the event to its parent element, is called Event Bubble.

What is a callback function and why will you use it?

Answer: Callback function is a type of function which can be passed into a different function as argument and can be used for more action.
